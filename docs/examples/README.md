# Examples

These are concrete alpha-phase walkthroughs built around the current codebase.

## Examples

- [local-qemu-smoke.md](local-qemu-smoke.md): build `chuboctl`, inspect the CLI, and run the main QEMU lane
- [helper-bundles.md](helper-bundles.md): fetch helper bundles and use `wonton`, `gyoza`, and `bao`

## Notes

- These examples are intentionally local-first.
- They assume alpha-level rough edges and prefer the smallest authoritative loop.
- On macOS, root-run QEMU remains the main validation path.
